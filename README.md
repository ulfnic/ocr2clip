# ocr2clip
Read text within a screen-grab and put it in your clipboard.

## Compatibility
X11 or Wayland based distributions

## Dependencies
- `tesseract`

#### X11:
- `maim` or `import`
- `xclip` or `xsel`

#### Wayland:
- `grim`
- `slurp`
- `wl-copy`

#### Optional for improved accuracy:
- `ffmpeg` or `ImageMagick`
